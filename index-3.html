// WHO'S BEHIND THE PANEL
About

I'm a senior IT support engineer who got tired of doing the same five-minute task for the fortieth time. Most of what's here started as a script written to fix one annoying Tuesday, then cleaned up enough to hand to someone else.

Nothing on this page is theoretical. Every port in the panel above has run in a real environment, against real data, with real consequences if it broke.

This isn't an agency and it isn't a product company. It's one engineer's working notebook, published in case the same problem is eating someone else's Tuesday too.
